Laporan Praktikum 7: Integrasi Bootstrap dan CSS Custom

Pendahuluan

Praktikum kali ini berfokus pada bagaimana mengintegrasikan framework Bootstrap dengan CSS custom untuk menciptakan tampilan yang unik dan profesional. Kita akan mempelajari cara menimpa style bawaan Bootstrap serta cara mengorganisir file CSS agar tetap rapi dan mudah dikelola.

Bootstrap: Framework CSS paling populer untuk membangun situs responsif dan mobile-first.
CSS Custom: Kode CSS buatan sendiri yang digunakan untuk menyesuaikan desain di luar standar framework.
Responsive Design: Pendekatan desain web yang membuat halaman web tampil baik pada berbagai perangkat.

Langkah Kerja

1. Persiapan Struktur File

Buatlah folder baru bernama laporan di dalam direktori proyek utama. Kemudian, buatlah file web7.html dan style-laporan.css di dalamnya.

[Tempat Screenshot: Struktur Folder Proyek]
2. Menghubungkan Bootstrap

Tambahkan link CDN Bootstrap 5 di bagian <head> file HTML Anda. Pastikan versi yang digunakan adalah yang terbaru untuk fitur yang lebih lengkap.

[Tempat Screenshot: Kode Header HTML]
3. Implementasi Container dan Card

Gunakan class .container dari Bootstrap dan buat container custom dengan background putih dan border-radius besar untuk memberikan kesan modern dan premium.
